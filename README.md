# OneAZ

OneAZ is a full-stack Learning Management System built using the MERN Stack. The platform enables educators and learners to manage courses, assignments, and educational resources efficiently.

## Features

- User Authentication
- Course Management
- Assignment Creation
- Assignment Submission
- Role-Based Access Control
- Responsive Design

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js

## Installation

### Clone the repository

git clone <repository-url>

### Install dependencies

Frontend:

cd client
npm install

Backend:

cd server
npm install

### Start the application

npm start
